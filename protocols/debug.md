# Debug Protocol

Use a structured loop instead of guess-and-patch behavior.

## Loop

1. Reproduce
2. Observe
3. Form a concrete hypothesis
4. Test the hypothesis
5. Fix the confirmed cause
6. Verify the regression is closed

## Rules

- Do not stack multiple speculative fixes at once.
- Keep notes on the confirmed root cause.
- If the issue is not reproducible, say so and reduce uncertainty before editing.
